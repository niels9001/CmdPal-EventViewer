# Privacy Policy

**Event Viewer for Command Palette**

Last updated: March 31, 2026

## Overview

Event Viewer for Command Palette is a local-only extension that reads Windows Event Logs and displays them within the Command Palette interface.

## Data Collection

This extension **does not collect, store, transmit, or share any personal data**. All event log data is read locally from your machine and displayed only within the Command Palette UI.

Specifically, this extension:

- Does **not** connect to the internet
- Does **not** send telemetry or analytics
- Does **not** use cookies or tracking technologies
- Does **not** store any data outside of your local machine
- Does **not** share data with any third parties

## Data Access

The extension reads Windows Event Logs using the standard `System.Diagnostics.EventLog` and `System.Diagnostics.Eventing.Reader` APIs. This data never leaves your device.

## Changes

If this privacy policy changes, the updated version will be posted in this repository.
